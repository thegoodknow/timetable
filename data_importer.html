Timetable Data Importer (HTML to JSON)

Automated Update Steps:

  1. The tool is **automatically** fetching your existing `timetable_data.json`.
  2. PASTE: Paste the raw HTML of the **new week's** timetable into the text area.
  3. CONVERT & DOWNLOAD: Click the button. The merged file will automatically download as **`timetable_data.json`**.
  4. **CRUCIAL MANUAL STEP:** You must manually **overwrite** the old `timetable_data.json` file in your GitHub repository with the newly downloaded file, and commit the change.
  5. The page will **automatically redirect** you to the timetable page to view the update.
